Verses Memorization helps you to memorize Bible verses by typing. May your heart and mind be filled with God's words, which you can put to good use in time of need.

How to get started?
Input your own favorite verses which you want to memorize.
Click the right arrow next to a verse reference, and start to type the content of the verse.
Hint: if you want to review the verse content while you type, you can click the left arrow next to the verse.

How to edit the reference and content of verses and tag names I input?
Double click on it. After you make a change, double click on the content to save the change.

How to add a tag to a verse?
Drag a tag to a verse.

What browsers can this app run in?
Any modern browsers that support HTML5.

Can this app work offline?
Yes. It doesn't need internet connection once it loaded. However, it cannot auto load ESV when input new verse.

Where is ESV got loaded from?
Scripture taken from The Holy Bible, English Standard Version. Copyright ©2001 by Crossway Bibles, a publishing ministry of Good News Publishers. Used by permission. All rights reserved. Text provided by the Crossway Bibles Web Service.

Will it have other versions of Bible auto loaded?
When other versions of Bible have their public API available. Please let Johann know if you find them.

Where is the data stored?
The data you input is stored completely in your local browser. The data persists as long as you do not clear your browser's localStorage memory.

How much data can it stored?
As much as the localStorage of your browser allows. Usually 5MB in a modern browser.

Does the app automatically sync data across browsers?
No, since the data is not stored in server. It's up to you to backup your data. You can manually sync data across other browsers.

How to manually sync data across browsers?
You can export data and download it to your local desktop, namely verses.txt and tags.txt (You can store these two files in some cloud service, such as dropbox). Then you can import them into a new browser. Be careful, the imported data will replace the current data of the new browser, so make sure that the data you are about to import is the most recent data.

What architect does this app use?
Client-side HTML5, CSS and Javascript. It's a single page app built with framework Angularjs, jQuery and Twiitter Bootstrap. It uses Sinatra as a simple backend web server. Icons from Glyphicons Free For source code of the app, please go to Github.
